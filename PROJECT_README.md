# 人情份子钱记录应用

一个专业的人情往来记录管理工具，帮助用户轻松管理送礼和收礼记录。

## 🎯 功能特色

- **📱 移动优先**：专为手机端优化的响应式设计
- **🔐 多用户支持**：用户注册登录，数据完全隔离
- **💰 礼簿管理**：记录送礼和收礼，支持联系人关联
- **📊 统计分析**：月度收支统计，数据可视化
- **📤 数据导出**：支持Excel格式导出，方便备份
- **📥 数据导入**：支持从Excel文件批量导入数据
- **🎨 美观界面**：现代化UI设计，操作简便

## 🚀 快速开始

### 环境要求
- Node.js 18+ 
- npm 或 pnpm

### 安装依赖
```bash
npm install
```

### 配置环境变量
复制 `.env.local` 文件并配置您的 Supabase 信息：
```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:5173 查看应用

- Excel格式导出

- 数据备份支持

## 🏗️ 技术架构

### 前端技术栈
- **React 18** + **TypeScript** - 现代化前端框架
- **Vite** - 快速构建工具
- **Tailwind CSS** - 实用优先的CSS框架
- **React Router v6** - 路由管理
- **Lucide React** - 图标库
- **React Hot Toast** - 通知提示

### 后端服务
- **Supabase** - 后端即服务平台
  - PostgreSQL 数据库
  - 用户认证服务
  - 实时数据同步
  - 行级安全策略

### 数据导入导出
- **SheetJS (xlsx)** - Excel文件处理

## 📱 移动端优化

- 响应式设计，适配各种屏幕尺寸
- 触摸友好的交互设计
- 底部导航栏，单手操作便利
- 卡片式布局，信息层次清晰

## 🔒 安全特性

- 用户数据完全隔离
- 行级安全策略（RLS）
- HTTPS传输加密
- 敏感信息本地加密

```bash
npm run build
```

支持一键部署到 Vercel 等现代化托管平台

## 🎯 使用说明

1. **注册账户**：使用手机号注册新账户
2. **添加记录**：点击底部"记账"按钮，填写礼簿信息
3. **管理联系人**：在联系人页面添加和管理您的联系人
4. **查看统计**：在首页查看月度收支统计
5. **导出数据**：在设置页面导出数据到Excel文件
